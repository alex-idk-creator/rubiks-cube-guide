# Rubik's Cube Guide

Русскоязычный статический сайт для изучения метода Фридрих / CFOP.

## Что внутри

- фронтальные обозначения движений: кубик смотрит на ученика одной гранью;
- простая главная страница с двумя действиями: начать сборку или понять формулы;
- полноценная светлая и тёмная тема с сохранением выбора;
- пошаговый маршрут CFOP: Cross, F2L, OLL, PLL;
- Cross-раздел с проверкой белых ребер по боковым центрам;
- F2L-библиотека из 42 учебных вариантов с сеткой, мастером поиска и отдельным экраном разбора;
- 7 OLL-случаев после уже собранного желтого креста;
- полный набор из 21 PLL-случая;
- рабочий режим `Начать сборку` с выбором этапа и подсказками по распознаванию;
- поиск и уровни изучения без лишних вкладок;
- SVG-схемы с цветными F2L-элементами, центрами и цветным слотом;
- пошаговый разбор выбранного случая: что найти, как держать, что делать, как проверить, с возвратом к месту в списке;
- уровни изучения: стартовые случаи, частые случаи и полный справочник.

## Как открыть

Открой `index.html` в браузере. Для GitHub Pages достаточно опубликовать репозиторий
из ветки `main` и выбрать корень проекта как источник Pages.
